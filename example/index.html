A Note About Styles

For simplicity's sake, no css dependency is included with this package-- just the angular.min.js and corresponding .map files. To make your CIs (Circle Images) behave more like standard images, add the following style somewhere in your CSS:

.ci-wrapper {
	display: inline-block;
	vertical-align: middle;
}
